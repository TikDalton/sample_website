Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the proess of making it, we'll learn how to make HTML tables via table and related tags.

The tables belowe don't include all HTML tags, but they do list the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header label a section of the page
div div divide sections of the page
ol ordered list create a list of ordered things (1,2,3 etc.)
ul unordered list create an unordered list (bulleted points etc.)

Inline Elements

Tag Name Purpose Example Result
em emphasised make emphasised text <em>techinal sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make a hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code displays pieces of markup or source code <code><em></code> em
span span groups and applies styles to inline elements <span></span> Something styled inline
li list defines a list item <li>Vengeful whale</li>
<li>Salty sailors</li>
<The names "Queequeg" and "Starbuck">
  • Vengeful whale
  • Salty sailors
